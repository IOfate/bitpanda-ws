# BitPanda WS

A lib to use BitPanda websockets.

Websocket API documentation: https://developers.bitpanda.com/exchange/?shell#websocket-api-overview

## Features

- [ ] Price ticks
  - [x] Subscription
  - [ ] Unsubscribe
- [ ] Candlesticks
  - [x] Subscription
  - [ ] Unsubscribe
- [ ] Order book
  - [ ] Subscription
  - [ ] Unsubscribe
- [ ] Market ticker
  - [ ] Subscription
  - [ ] Unsubscribe
