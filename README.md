# BitPanda WS

A lib to use BitPanda websockets.

Websocket API documentation: https://developers.bitpanda.com/exchange/?shell#websocket-api-overview

## Features

- [x] Price ticks
  - [x] Subscription
  - [x] Unsubscribe
- [x] Candlesticks
  - [x] Subscription
  - [x] Unsubscribe
- [ ] Order book
  - [ ] Subscription
  - [ ] Unsubscribe
- [ ] Market ticker
  - [ ] Subscription
  - [ ] Unsubscribe
